Projeto de exemplo usando AngularJS

Projeto client-side simples, usando AngukarJS, nodeJS e bower.

Configurações necessárias::

  $ sudo apt-get install nodejs
  $ sudo apt-get install npm
  $ sudo npm install -g bower

  Configuração de variaveis de ambiente e caminho do nodejs
  $ export BOWER_HOME=/usr/local/lib/node_modules/bower

  $ export PATH=$PATH:$BOWER_HOME/bin
  $ sudo ln -s /usr/bin/nodejs /usr/bin/node

  $ mkdir ~/myproject
  $ cd myproject

  Configuração do projeto (package.json)
  $ npm init 
  $ npm install
  $ bower install
